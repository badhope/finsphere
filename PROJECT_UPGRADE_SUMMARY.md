## 🎮 使用方式汇总

### 方式一：Web图形化管理（推荐）
```bash
python web_manager.py
# 浏览器自动打开 http://localhost:8080
```

### 方式二：新手友好启动
```bash
# Windows用户双击运行
beginner_start.bat
```

### 方式三：命令行操作
```bash
# 环境检测
python scripts/detector.py

# 自动配置
python setup.py

# 启动开发服务器
npm run dev
```

## 🚀 部署和分发建议

### 📦 本地开发环境
```bash
git clone [项目地址]
cd finsphere
python beginner_start.bat  # Windows
# 或
./beginner_start.sh        # Linux/macOS
```

### ☁️ 云端部署
```bash
# Docker部署
docker build -t finsphere-pro .
docker run -d -p 80:80 finsphere-pro

# 传统服务器部署
npm run build
# 部署dist目录到Web服务器
```
