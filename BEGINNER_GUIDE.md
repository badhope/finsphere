# 🌟 FinSphere Pro 新人友好使用指南

## 🎯 写给新手的贴心指引

欢迎来到 FinSphere Pro！这是一个专门为新手设计的企业级金融管理平台。即使你是编程新手，也能轻松上手。

## 🚀 三步快速开始

### 第一步：环境准备
```bash
# 双击运行（Windows用户）
双击 setup.bat

# 或者使用命令行
python setup.py
```

### 第二步：启动项目
```bash
# 方法一：使用Web管理面板（推荐新手）
python web_manager.py

# 方法二：使用命令行
npm run dev
```

### 第三步：开始开发
打开浏览器访问 `http://localhost:3000`

## 🤔 常见问题解答

### Q: 我完全不懂编程，能使用吗？
**A**: 当然可以！我们的Web管理面板就像手机APP一样简单易用。

### Q: 需要安装什么软件？
**A**: 只需要Python，其他软件都会自动安装。

### Q: 项目运行后在哪里看效果？
**A**: 浏览器会自动打开 `http://localhost:3000`

### Q: 数据会保存吗？
**A**: 目前数据保存在浏览器内存中，刷新页面会重置。

### Q: 如何停止项目？
**A**: 在命令行按 `Ctrl+C` 或关闭浏览器窗口
